# شات زهراء - تعليمات البناء

## لبناء التطبيق مجاناً عبر GitHub:

1. **حمّل هذا المجلد** كاملاً إلى جهازك.
2. اذهب إلى [GitHub](https://github.com/new) وأنشئ مستودعاً جديداً باسم "ShaatZahraa".
3. ارفع كل ملفات المشروع إلى المستودع.
4. اذهب إلى تبويب **"Actions"** في المستودع.
5. اختر **"Build Android APK"** من القائمة اليسرى.
6. اضغط على **"Run workflow"**.
7. انتظر حوالي 2-3 دقائق حتى تكتمل العملية.
8. ستجد ملف APK في قسم **"Artifacts"** أسفل الصفحة.
9. حمّل الملف وثبّته على هاتفك مباشرة.

**هذه الطريقة أسهل بكثير من استخدام Android Studio!**
